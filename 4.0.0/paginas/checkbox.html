Checkbox

Elemento de html que permite seleccionar una o más opciones de varias posibles tildando las opciones seleccionada. Bodystyle provee un conjunto de estilos que permiten mostrar a estos elementos con cierto formato y mejorar la experiencia del usuario.

Checkbox Group

La forma en la que se podrán aplicar los estilos a este elemento es a través de un contenedor que envuelva al elemento
Como se puede ver en el ejemplo hay un contenedor padre con la clase .check-radio-grupo que envuelve a un label con la clace .cd-azul y a un input del tipo checkbox con la clase checkbox.

Colores

La librería dispone de varios colores para este elemento determinado por las necesidades del desarrollador en un momento determinado.
