Colecciones

En este apartado vamos a ver un elemento muy utilizado en todo sitio web para agrupar elementos de características similares, las calecciones son un forma muy intuitiva de distribuir los elementos de manera tal de mejorar la expreriencia del usuario y el recorrido a través de la aplicación o sitio web en desarrollo.

Colecciones por default

Vamos a ver como generar una colección con los estilos básicos que permitirán agrupar un conjunto de enlaces para la navegación.
La clase utilizada como contenedor es .list-container y dentro de esta .list-intem que envuelve a cada enlace de la lista.
