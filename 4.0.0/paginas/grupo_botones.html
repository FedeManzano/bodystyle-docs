Grupo de Botones

Al igual que el partado anterior (Botones) vamos a ver otro elemento que permite el agrupamiento de estos elementos de una manera más compacta y definida en tres tamaños diferentes.

Grupo de Botones Tamaños

La librería cuenta con tres tamaños (LG, MD, SM) para la definición de estos elementos.
La clase utilizada es btn-group / btn-grupo y la misma envuelve a los botones que se deseen agrupar.
