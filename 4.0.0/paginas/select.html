Select

El elemento de esta sección es el Select de html con un conjunto de estilos visuales que permiten una mejor visualización del elemento a través de estilos CSS y la lógica de javascript para el comportamiento dinámico.

Select Iniciación

Para poder utilizar este elemento tenemos que tener inicializado el módulo que se encarga de esta tarea. Es necesario agregar antes del cierre de la etiqueta body de html el archivo bodystyle.min.js.

Para que todo funcione correctamente asignarle a etqueta div con la clase .select-mod-dark agregarle un id seleccionado por el desarrollador y luego pasar este ID a través del parámetro de la función BS.SelectInit().iniciar("#IDSeleccionado") no olvidarse el numeral delante del ID.
Tener en cuenta que un ID es único para cada elemento del DOM que tenga uno, no puede repetirse.

Temas Select

La librería cuenta con dos temas. El primero es el tema White y el segundo es el tema Dark, en ambos casos la funcionalidad es la misma. 

        
        

            
        


        
        

            
        

    
        BS.SelectInit().iniciar("#s_white", "borde"); // ID coincidente con lo definido en el HTML
        BS.SelectInit().iniciar("#s_dark", "borde"); // ID coincidente con lo definido en el HTML

Efectos Hover

La librería cuenta con opciones para establecer el efecto de selección de opción en el select. Puede ser borde o fondo.
En el caso de borde lo vimos en los ejemplos anteriores con el segundo parámetro enviado ahora veremos el segundo (fondo). 

        
        

            
        

    
        BS.SelectInit().iniciar("#s_fondo", "fondo"); // ID coincidente con lo definido en el HTML
