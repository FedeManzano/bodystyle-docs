Select

El elemento de esta sección es el Select de html con un conjunto de estilos visuales que permiten una mejor visualización del elemento a través de estilos CSS y la lógica de javascript para el comportamiento dinámico.

Select Iniciación

Para poder utilizar este elemento tenemos que tener inicializado el módulo que se encarga de esta tarea. Es necesario agregar antes del cierre de la etiqueta body de html el archivo bodystyle.min.js
