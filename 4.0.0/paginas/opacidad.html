Opacidad

Conjunto de clases CSS predefinidas para definir el nivel de visibilidad de un determinado elementos dentro del DOM.
Si un elemento tiene una opacidad menor a uno la visisbilidad va a tener un nivel de transparencia acorde con el valor definido.

Opacidad de un Elemento

La clase para llevar a cabo esta tarea es opa-[nivel] donde nivel es un valor de 1 a 10, de manera tal que 1 es equivalente a 0.1 de la propiededad CSS opacity.

La propiedad de CSS es opacity: [0-1], en el ejemplo anterior si utilizamos la clase opa-0, entonces el elemento no se verá en pantalla. Caso contario si utilizamos el parámetro 10 / opa-10 el elemento se verá en su totalidad si ningún nivel de transparecia.
