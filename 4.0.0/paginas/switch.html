Switch

Elemento Html que posee dos estados ON/OFF permite definir que hacer en función de una desición booleana que permite tomar caminos diferentes en función de la selección del usuario.

Muestra del Switch

En este apartado se muestra un ejemplo de un Switch en un determinado color.
Para el funcionamiento de este elemento es necesario envolver a todo el conjunto en u div con la clase .switch-grupo y luego, dentro del div contenedor agregamos input del tipo checkbox, luego agregar un label con el atributo for = (id del input checkbox).
