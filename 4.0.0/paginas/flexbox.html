Flexbox

En este apartado vamos a ver y analizar una de las funcionalidades más importantes de CSS 3, que permite el aliniamiento horizontal y vertical de los elementos con la facilidad que versiones anteriores de CSS no tenian, desde el punto de vista de la librería se dispondrán de clases que permitirán utilizar estas funcionalidades sin necesidad de archivos de estilos y aplicándolas directamente en el DOM de html.

Flexbox en CSS3

En CSS3 se utiliza la propiedad display: flex que por defecto acomoda a los elementos de forma horizontal uno pegado al otro. 

    .flex-box {
        display: flex;
    }

Como se puede observar se mantienen un contenedor pegado al otro sin ningún tipo de distancia entre ambos y alineados a la izquierda.
Para centrar los contenedores es necesario valerse de otra propiedad que acompaña a la anterior justify-content: center. 

    .flex-box {
        display: flex;
        justify-content: center;
    }

    .flex-box {
        display: flex;
        justify-content: space-around;
