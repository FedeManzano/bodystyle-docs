Show Code

Dependencia de Bodystyle para colorear el código de programación mostrado en páginas web.
La dependencia busca colores representativos de cada lenguaje para que sea más llamativo y entendible en la web. Los lenguajes incluídos son:

  • C
  • Java
  • JS
  • CSS
  • Html

Podemos descargar la dependencia por separado a través del siguiente enlace: Show-Code 1.1.1.
Si bien se encuentra en desarrollo y las funcionalidades que aporta son limitadas sirvió para mostrar código de toda esta documentación a través de los cuadros donde se encuentran los códigos de ejemplo.
