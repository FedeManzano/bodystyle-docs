Modal

El Modal es una herramienta muy utilizada en todo tipo de web, ya que captura y centraliza la la atención del usuario para gestionar acciones y anular de manera momentanea todos los demás elementos. Se basa en una ventana emergente que contiene por lo genera formularios, tabs y accesos a determinadas funciones encapsuladas.

Modal Inicialización

Para pode inicializar el módulo que gestiona este elemento tenemos que disponer de la lógica de la librería agregada al proyecto, esto se puede hacer de diversas maneras, si aún no saben como realizar esto les recomiendo ver Get Started donde se mostrarán como instalar la librería en el proyecto en desarrollo. Luego hay que inicializar el módulo a través de la función que se muestra a continuación. 

    BS.ModalInit() // Con esta linea inicializamos el módulo
