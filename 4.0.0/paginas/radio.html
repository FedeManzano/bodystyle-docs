Radio

Elemento de html que permite seleccionar una opción de un conjunto para definir preferencias o dar una respuesta a una pregunta o seleccionar un medio de pago.
Bodystyle brinda estilos para poder definir estos elementos a través de grupos de elementos agrupados.

Radio Button

Aquí un ejemplo de un radio button con la librería. Se mostrarán seis radio button pertenecientes al Grupo 1 lo cueal permitirá seleccionar solo uno de ellos.
Es importante definir el grupo al cual pertenece el conjunto de radio buttons.
En este caso para poder utilizar estos elementos hay que envolver al label y al input type radio dentro del contenedor .check-radio-grupo, luego al label agregarle la clase .rd-[rojo | verde | negro | gris | blanco] junto con el input type radio que tiene la clase .radiobtn.
