Padding

Distancia que existe entre los límites de un elemento y su contenido, en este apartado vamos a aprender a como gestionar este espacio con las clases provistas por la librería para todos los tamaños de pantalla.

Padding General

Lo primero a evaluar es el padding general perteneciente a todos los tamaños de pantalla por igual sin alteraciones. La clase utilizada para esta tarea es p-[em] siendo em el tamaño del padding correspondiendose con 1em = 16px y el parámetro definido en uno son 0.5em = 8px.

Padding Medidas

Se pueden utilizar las clases de padding para que el mismo sea diferente en cada dispositivo donde se muestre el elemento.

p-0 pl-m-5 pr-s-5 
    

        

            p-0 pl-m-5 pr-s-5
En el ejemplo anterior se tiene un padding de 0 para equipos con pantalla superior a 1030px, mientras que para equipos menores a 1030px se le añade un padding de 2.5em a la izquierda y para equipos con menos de 770px se le añade un padding derecho de 2.5em.

Padding X e Y

La librería permite establecer el padding tanto arriba y abajo como a los lados con las clases px-[em] y py-[em].
Incluye también las clases que permiten definir esta propiedad en diferentes tamaños de pantalla.

.px-5
.py-5
En el ejemplo anterior se introduce un padding arriba y abajo, y en otro contenedor un padding a la izquierda y a la derecha de 2.5em.
