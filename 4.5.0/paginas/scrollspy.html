ScrollSpy

Elemento que sirve como guia para el usuario dentro una página, está conformado por un cojunto de enlaces que vincularán el contenido con la guia, y de esta forma le dará al usuario accesos directos a partes específicas de la página.

Inicialización

La inicialización de este módulo no solo permitirá brindar la lógica de funcionamiento sino que le brindará al elemento un conjunto de estilos propios de Bodystyle de manera tal que el desarrollador podrá personalizar el ScrollSpy.

Parámetro Valor Descripción Default Obligatorio
ancho 1-100% Ancho del scrollspy medido en % 16 NO
tamFuente 1-30px Tamaño de la fuente en PX 16 NO
colorBorde fd-[color] Clases de colores de fondo de la librería Ver Doc Solo las clases que comienzan con fd-[color] bg no incluidas aún fd-azul NO
alturaBorde 1-100 Altura del borde que se agrega de manera dinámica 3 NO
separacion 1-n PX Separación del scrollspy de la parte superior de la pantalla 100 NO
colorSeleccionado c-[color] Color del texto del enlace cuando es seleccionado (hexadecimal) #000 NO
colorNoSeleccionado c-[color] Color del texto del enlace cuando no está seleccionado (hexadecimal) #666 NO
Si cualquiera de los parámetros de inicialización descriptos en la tabla anterior no es correcto bodystyle le imprimirá un mensaje en la consola del navegador con el código del error y una descripción detallada del mismo. Para más información consultar: Gestor de Errores

    /**
     * Todos los parámetros están validados
     * cualquier error que ocurra pueden consultar la consola del 
     * navegador, ahí se les informará cual fué el error y podrán corregirlo.
     */
    let conf = {
        ancho: 15, // Ancho en porcentaje del scrollspy
        tamFuente: 17, // Tamaño de la fuenta
        colorBorde: "fd-bodyui", // color del borde
        alturaBorde: 30, // Altura del elemento dinámico borde
        separacion: 150, // Separación con respecto al inicio de la pantalla propiedad TOP
        colorSeleccionado: "#fff", // Color del enlace seleccionado
        colorNoSeleccionado: "#ccc" // Color del enlace no seleccionado
    }
    
    // Inicialización con la conf
    BS.ScrollSpyInit(conf)
