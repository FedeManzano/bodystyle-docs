Lista Fixed

Listas que se posicionan a la derecha o la izquierda de la pantalla y permanece en esa posición durante toda la pantalla, de manera tal que acompaña el usuario durante todo el recorrido de la página.
A la derecha de esta página hay una muestra del elemento de esta sección.

Inicialización

Para inicializar el elemento es necesario disponer del archivo bodystyle.min.js agregado en el proyecto en desarrollo, para hacer esto consultar Get Started de la documentación.
Luego hay que inicializar el módulo de la librería a través del objeto BS que permite el acceso a todos los módulos de la librería.

Parámetros del Módulo

Atributo Valores Descripción Obligatorio Default
Contexto ID que envuelve a toda la lista Importante parámetro porque permite configurar el elemento, es importante definir un ID unívoco para poder manejar ele elemento. SI vacio (no funciona)
fondoItem (fd-[color]|bg-[color]) Permite definir el color de fondo de los elementos de la lista. Son las clases de la librería que permiten configurar el color de fondo. Ver Colores NO fd-gris-ve-o
colorTexto (c-[color]) Permite definir el color de texto de los elementos de la lista. Son las clases de la librería que permiten configurar el color del texto. Ver Texto NO c-white
altura Valores entero positivo Altura respecto el eje Y en PX donde queremos que aparezca la lista. NO 150

Ejemplo en esta Página

Código utilizado para implementar el elemento que se muestra a la derecha en esta misma página.
Es importante envolver toda la lista en un contenedor con un ID único para cada lista instanciada, luego utilizar las clases .lista-float-der o .lista-float-izq que permitirán posicionar la lista a la derecha o a la izquierda con respecto a la pantalla. El paso siguiente es envolver a los items de la lista en un contenedor con la clase .lista-item como se muestra en el código de abajo. Por último agregamos los botones de abrir y cerrar la lista con las clases .abrir y .cerrar.

    // CDN : https://raw.githack.com/FedeManzano/bodystyle/refs/heads/master/dist/js/bodystyle.min.js

    // 1 OPCIÓN ESTÁTICA

    // Inicializa el elemento
    // cuando estos están dispuestos de manera estática dentro del DOM

    // Notese que el ID de contexto es igual al definido en el HTML pero con el numeral delante
    BS.ColeccionFlot().iniciar({contexto:"#lis1", fondoItem:"bg-pink-d-1", colorTexto:"c-blanco", altura:300} )

     // 2 OPCIÓN DINÁMICA
    // -----------------   
    
    /*
        En proyectos donde se tabaja a través de injección de elementos a través 
        de JavaScript de manera dinámica, y el elemento que se estudia en esta sección se 
        ingresa de esta manera, es necesario aplicar un retraso en la inicialización del 
        módulo con el siguiente codigo.
    */

    //Si los elementos que conforman esta funcionalidad son dinámicos 
    // Agregados a través de JS 
    window.onload = () => {
        setTimeout( () => {
            BS.ColeccionFlot().iniciar({contexto:"#lis1", fondoItem:"bg-pink-d-1", colorTexto:"c-blanco", altura:300} )
        }, 100); 
    }
