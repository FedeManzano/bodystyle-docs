Breadcrumbs

Elemento simple compuesto sólo por clases CSS y que permite brindale una guia al usuario para saber en que parte del proceso se encuentra. Por ejemplo cuando el usuario se encuentra completando un formulario de varias etapas, este elemento le permite al usuario conocer la etapa en la que se encuentra.

Breadcrumbs Oscuro

La librería tiene dos temas uno claro y uno oscuro para ajustarse a todos los sitios que implementen diferentes temas en su diseño.
