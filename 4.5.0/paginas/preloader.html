Preloader

Elemento sencillo que busca entretener al usuario cuando está a la espera de un recurso o la carga de cierto módulo de la aplicación tarda algunos segundos en cargarse.

Tamaños

La librería cuanta con tres tamaños para este tipo de elementos, (sm | md | lg) donde md no hace falta ponerlo porque es el valor por default.
Para utilizar este elemento tenemos que agregar a una etqueta .preloader-[TAM] donde TAM es el tamaño del preloader, luego en el mismo elemento la clase pre-[color] donde color es el color del preloader.

Colores

Existe algunos colores para definir estos elementos, los mismos se detallan en la siguiente lista.

  • .pre-rojo
  • .pre-verde
  • .pre-azul
  • .pre-negro
  • .pre-gris
  • .pre-blanco
