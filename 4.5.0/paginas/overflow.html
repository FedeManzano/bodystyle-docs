Overflow 4.5.0

Compuesto por tres clases que permiten definir el la visibilidad de un contenedor por fuera de su contenedor padre.

Overflow Clases

Se utilizan tres clases que permiten tres estados diferentes de visibilidad. La propiedad CSS utilizada es overflow. Las clases de Bodystyle implicadas son:

  • .ov-hidden
  • .ov-visivle
  • .ov-scroll
    •     /* No permite que el contenedor hijo salga de los límites del contendor padre.*/
    .ov-hidden {
        overflow: hidden !important;
    }
    
    /* Permite que el contenedor hijo salga de los límites del contenedor padre.*/
    .ov-visible {
        overflow: visible !important;
    }

    /*
        No permite que el contenedor hijo no salga del contenedor padre pero 
        introduce un scroll en el elemento para que se vea el contenido.
    */
    .ov-scroll {
        overflow: scroll !important;
    }

    Muestra .ov-hidden

    Vamos a realizar un ejemplo sobre los bordes redondeados, cuando redondeamos los bordes de un contenedor padre y el mismo contienen un elemento de las mismas dimensiones no se verán los bordes redondeados ya que por defecto la propiedad es visible.
    Notese que el contenido del elemento interno no permite ver los bordes redondos del ejemplo, ahora bien que sucede si aplicamos la clase ov-hidden.
    En este ejemplo se puede observar la utilidad del elemento contenedor limita la visibilidad del contenedor hijo, ahora se pueden ver los bordes redondeados.
