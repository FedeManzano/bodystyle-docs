Alertas

Son un elemento simple que se puede utilizar cuando se necesite resaltar o aclarar una cuestión que haya quedado poco clara en el texto añadiendo información, enlaces o recomendaciones útiles para el usuario.
Desde la versión 4.0.0 se añadieron alias que permiten nombrar las clases de las alertas en el lenguaje inglés.

La librería cuenta con dos tipos de alertas, la primera es la utiliza un borde de color del lado izquierdo que es la que utilizaremos en este apartado.

Las clases utilizadas para definir este tipo de alerta es alert-op-[color] donde color puede ser:
  • red / roja
  • green / verde
  • black / negra
  • grey / gris
  • white / blanca
  • blue / azul
