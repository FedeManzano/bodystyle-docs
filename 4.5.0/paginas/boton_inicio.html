Botón Inicio

Elemento simple y dinámico que le permite al usuario volver al inicio de la página. El botón se muestra cuando el scroll se posiciona a 100px del inicio de la página.

Inicialización

Para inicializar este elemento hay que disponer del archivo bodystyle.min.js agregado al proyecto y luego iniciar el módulo correspondiente al botón de inicio.
Esta documentación posee el botón en la parte inferior derecha de cada página y sirve como muestra de como se vería el elemento en un sitio en desarrollo. 

    /*
            Dentro de etiquetas script de html debajo del archivo bodystyle.min.js 
            agregar el siguiente código. 
    */
    BS.BotonInicioInit() // Automáticamente aparecerá el Botón
