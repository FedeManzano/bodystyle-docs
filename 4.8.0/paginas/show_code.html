Show Code

Dependencia de Bodystyle para colorear el código de programación mostrado en páginas web.
La dependencia busca colores representativos de cada lenguaje para que sea más llamativo y entendible en la web. Los lenguajes incluídos son:

  • C
  • Java
  • JS
  • CSS
  • Html

Podemos descargar la dependencia por separado a través del siguiente enlace: Show-Code 1.1.1.
Si bien se encuentra en desarrollo y las funcionalidades que aporta son limitadas sirvió para mostrar código de toda esta documentación a través de los cuadros donde se encuentran los códigos de ejemplo.

Show Code Inicialización

Para poder utilizar las funcionalidades provistas por Show-Code es necesario agregar a nuestro proyecto el archivo de javascript de la librería bodystyle.min.js este proceso de puede realizar de varias maneras como se describe en Get Started para llevar a cabo los ejemplos que siguen a continuación lo haremos a través de su CDN. 

    // CDN : https://raw.githack.com/FedeManzano/bodystyle/refs/heads/master/dist/js/bodystyle.min.js

    // 1 OPCIÓN ESTÁTICA
    // ------------------

    // Inicializa el elemento
    // cuando estos están dispuestos de manera estática dentro del DOM
    BS.CodigoCInit(); Si necesitamos el código de C
    BS.CodigoJavaInit(); Si necesitamos el código de Java
    BS.CodigoJsInit(); Si necesitamos el código de JavaScript
    BS.CodigoHtmlInit() Si necesitamos el código de Html
    BS.CodigoCssInit() Si necesitamos el código de CSS

    /*******************************************************************************/

    // 2 OPCIÓN DINÁMICA
    // -----------------   
    
    /*
        En proyectos donde se tabaja a través de injección de elementos a través 
        de JavaScript de manera dinámica, y el elemento que se estudia en esta sección se 
        ingresa de esta manera, es necesario aplicar un retraso en la inicialización del 
        módulo con el siguiente codigo.
    */

    //Si los elementos que conforman esta funcionalidad son dinámicos 
    // Agregados a través de JS 
    window.onload = () => {
        setTimeout( () => {
            BS.CodigoCInit(); Si necesitamos el código de C
            BS.CodigoJavaInit(); Si necesitamos el código de Java
            BS.CodigoJsInit(); Si necesitamos el código de JavaScript
            BS.CodigoHtmlInit() Si necesitamos el código de Html
            BS.CodigoCssInit() Si necesitamos el código de CSS
        }, 100); 
    }

Luego hay que agregar en la cabecera de la página los estilos CSS utilizados por Show Code, los mismos están disponibles desde la descarga directa Show-Code 1.1.1 o a través de su CDN que se mostrará a continuación. Es importante aclarar que la librería cuenta con dos temas para agregar, el tema claro y el tema oscuro, hay que elegir cual de los dos se utilizará.

Codigo C

Para colorear el codigo de C tenemos que añadirle a la etiqueta pre la clase cod-c y luego dentro de la misma colocar el código que se desee mostrar. 

    #define ERROR -10
   
   
    /* Declaramos una función */
    int suma(const int n1, const int e2)
    {
        // Devuelve la suma realizada
        return n1 + n2;
    }

    /** Función que utiliza punteros */
    int sumaPunteros(const int * n1m const int * n2)
    {
        if(!n1 && !n2)
            return *n1 + *n2;
        else 
            return ERROR;
    }

    int main()
    {
        int var1 = 3;
        int var2 = 5;

        printf("La suma es %d: ", var1 + var2);
        printf("La suma es %d: ", suma(var1, var2));
        printf("La suma es %d: ", sumaPunteros(&var1, &var2));
        return 0;
    }

Codigo Java

Para este código se utilizará la clase cod-java asignada a la etqueta html pre para mostrar el código. 

        public class Alumno extends Persona 
        {
            private String nroDocumento;
            private String nombre;
            private String apellido;

            /**
                Constructor parametrizado de la clase alumno concreta
                @param (nroDocumento String)
                       (nombre String)
                       (apellido String)
                @see Persona
            */
            public Alumno(String nroDocumento, String nombre, String apellido){
                super();
                this.nroDocumento = nroDocumento;
                this.nombre = nombre;
                this.apellido = apellido;
            }

            /**
                Devuelve el nroDocumento del Alumno.
            */
            public String getNroDocumento()
            {
                return this.nroDocumento;
            }

            public String getNombre()
            {
                return this.nombre;
            }

            public String getApellido()
            {
                return this.apellido;
            }


            public static void main(String[] args)
            {
                Alumno a = new Alumno("25444444", "Federico", "Manzano");

                System.out.println("Alumno: " + a.getNroDocumento() + " " + a.getNombre() + " " + a.getApellido() );
            }
        }

Código CSS

El código css se hace de igual manera con la clase cod-css 


        /* 
            Clase que aplica el flexbox centralizado
        */
        .clase-ejemplo {
            display: flex; 
            justify-content: center;
            align-item: center;
            text-align: center;
            font-size: 20px;
        }

        /* Clses que empiezan con el prefijo fd-*/
        .ejemplo2[class^=fd-] {
            position: absolute;
        }

        /* selector por ID*/
        #titulo {
            display: flex;
        }
